# AI Agent Guide

This folder is an agent-facing map of the repository.

It exists to help coding agents understand:
- what this project is
- which parts are user-facing vs content/data tooling
- where major systems live
- how recent UI patterns are structured
- how to make changes without breaking established behavior

Use this folder as the first stop before editing the codebase.

Recommended reading order:

1. `01-project-purpose.md`
2. `02-repository-structure.md`
3. `03-developer-experience.md`
4. `04-shared-project-page-system.md`
5. `05-researcher-and-landing.md`
6. `06-resume-pipeline.md`
7. `07-engineering-conventions.md`
8. `08-common-change-guide.md`

If the task is only about resume generation, start with:
- `06-resume-pipeline.md`

If the task is only about project pages, start with:
- `04-shared-project-page-system.md`

If the task is only about the main developer page, start with:
- `03-developer-experience.md`
